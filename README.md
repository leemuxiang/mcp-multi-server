# MCP Multi-Service Server

这是一个基于 Model Context Protocol (MCP) 的多业务服务端项目，集成了 TodoList、计算器和地质工程计算功能。

- **[架构文档](./docs/architecture.md)**: 系统分层架构与数据流向。
- **[测试文档](./docs/testing.md)**: 单元测试与协议测试说明。

- **[SKILL 描述](./SKILL.md)**: 供大模型参考的工具使用指南。

## 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:3000` 即可进入可视化测试界面。
